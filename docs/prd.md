# 外贸工作记录工具需求文档

## 1. 应用概述

### 1.1 应用名称
外贸工作记录工具

### 1.2 应用描述
专为外贸行业设计的专业信息回溯工具,帮助业务人员快速记录和查找多环节业务信息,实现以事找人、以价找时、以样品找反馈的精准回溯需求,同时支持个人护肤习惯记录及报价格式自动生成。

## 2. 核心功能

### 2.1 快捷记录窗
- 全局快捷键呼出(如 Alt+Shift+Z)
- 极简输入框,仅包含时间戳(自动)和文本输入框
- 时间戳格式:YYYY-MM-DD HH:mm
- 点击时间区域弹出时间选择器,支持手动修改时间
- 支持可选标签勾选
- 5秒内完成一条记录的录入
- 标签勾选区右侧增加「+」图标
- 点击「+」图标弹出极简输入框,输入标签名→回车即创建并自动勾选
- 整个过程不离开记录窗

### 2.2 标签预设库
- 用户可自定义常用标签分类
- 首次启动预置行业默认标签:客户、品名、业务类型、运费、反馈、护肤
- 用户可对预置标签进行删除或新增
- 每条记录可勾选1-2个标签
- 仅勾选,不手打标签名称

### 2.3 时间轴浏览与检索
- 主界面展示倒序时间轴列表
- 顶部提供标签筛选下拉框,支持多选
- 顶部增加全局关键词搜索框,对所有记录的文本内容进行全文检索
- 筛选后仅显示相关记录
- 每条记录包含完整的关键词信息
- 每条记录支持右键菜单(或悬停出现图标):编辑、删除
- 编辑时时间戳默认可修改,点击时间区域弹出选择器
- 编辑窗口增加选项:□ 保留原始记录时间(默认不勾选)

### 2.4 纯文本本地存储
- 存储格式:所有记录保存为CSV格式,文件名自动生成如 2024-12-11.csv
- 存储位置:用户首次运行时自主选择根文件夹,后续按日期自动分文件写入
- 主界面底部常驻显示当前存储路径,点击可打开文件夹
- 文件为标准CSV,用户可随时用Excel打开进行排序、筛选、打印等操作
- 隐私承诺:永不联网、永不云端、永不收费

### 2.5 快速复制(可选)
- 时间轴列表每条记录右侧增加「📋」图标
- 点击后直接复制该条记录的完整文本内容到剪贴板
- 无需进入编辑页

### 2.6 报价格式生成器
- 页面名称:报价格式生成器
- 输入格式:商品名称+规格型号+数量+包装方式+单价
- 示例输入:汉堡HRM95 20吨小柜吨包价格500
- 输出格式:CIF City: USDXXX/MT, XXXMT for 1X20'FCL, packing in XXXkg big bags with pallets
- 示例输出:CIF Hamburg: USD500/MT, 20MT for 1X20'FCL, packing in 1000kg big bags with pallets
- 自动识别城市名称(如汉堡识别为Hamburg)
- 自动识别数量单位(如20吨识别为20MT)
- 自动识别包装方式(如小柜吨包识别为20'FCL和1000kg big bags)
- 支持一键复制格式化结果

## 3. 交互流程

### 3.1 首次启动
- 首次启动时不弹出设置页,应用直接进入托盘运行状态
- 标签库预置默认标签,用户第一次打开记录窗即可直接勾选
- 设置入口放在托盘右键菜单或主界面,需要时再去修改
- 显示全局快捷键说明
- 应用常驻系统托盘

### 3.2 日常记录
- 用户按下快捷键,快速呼出记录窗口
- 光标自动定位在文本输入框
- 输入一句话记录关键信息(如:美国A寄样325目水洗,FedEx单号123,运费450,预计反馈2周 或 洗脸,使用兰蔻洁面乳)
- 下方勾选相关标签(客户、品名、业务类型、护肤等)
- 回车或点击保存后,窗口自动关闭
- 整个过程控制在5秒内完成

### 3.3 回溯查找
- 双击系统托盘图标打开主界面
- 在顶部筛选栏选择标签(如客户=美国A,业务类型=寄样 或 标签=护肤)
- 在顶部搜索框输入关键词进行全文检索
- 列表自动过滤并展示相关记录
- 用户可直接查看历史记录中的完整关键词信息
- 复制所需信息即可使用

### 3.4 报价格式生成
- 用户点击报价格式生成器页面
- 在输入框中输入报价相关信息
- 系统自动识别并生成标准格式的报价文本
- 用户可直接复制生成的结果使用

## 4. 边界与演进

### 4.1 明确不做的功能
- 不自动抓取窗口标题
- 不自动生成日报草稿
- 不联网、不协作、不共享

### 4.2 后续扩展方向
- 支持附件上传(如快递单号链接、聊天记录截图、PDF检测报告)
- 支持标签层级分类管理
- 完善城市名称识别库
- 支持更多包装规格识别
